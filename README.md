農 Sight の管理画面です。
仮説検証をしながらすぐにデータを修正することができます。

# 開発に参加するまでの流れ

以下の手順を踏んで開発に参加しましょう。

## 1. `git clone`

以下のコマンドを実行してクローンします。

```shell
git clone https://github.com/nohsight/admin_dashboard.git
```

## 2. `npm install`

npm でパッケージをインストールします

```shell
#パッケージをインストール
npm install
```

## 3. `npm run dev`

`npm run dev`でローカルサーバーを立ち上げます。

```shell
npm run dev
```
